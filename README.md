PMAY Policy Clarification Assistant
Independent Public Policy Portal
About
PMAY Policy Clarification Assistant is a structured informational website designed to explain the Pradhan Mantri Awas Yojana (PMAY) in clear, accessible language.
The platform presents:
•	Plain-language policy explanations
•	Eligibility categories and subsidy structure
•	Urban vs Rural comparison
•	Frequently asked concerns and clarifications
•	Transparency dashboard
•	Data highlights
•	Eligibility checker
•	Methodology and disclaimer sections
This project is built as a fully responsive HTML5 website with embedded CSS and accessibility-focused design.

Objective
The objective of this portal is to provide:
•	Clear interpretation of policy guidelines
•	Structured public understanding
•	Transparent presentation of information
•	Search-friendly and accessible content architecture
This is an independent informational portal and is not affiliated with any government authority.
